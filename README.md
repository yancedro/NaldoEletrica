# Naldo Elétrica & Reformas ⚡

Landing Page profissional desenvolvida para a empresa **Naldo Elétrica**, focada em serviços de eletricidade residencial/predial, reformas civis, gesso e pintura no Rio de Janeiro.

## 🚀 Tecnologias Utilizadas
* HTML5
* Tailwind CSS (via CDN)
* Font Awesome (Ícones)
* Google Fonts (Montserrat e Open Sans)

## 📱 Funcionalidades
* Design 100% Responsivo (Mobile First)
* Galeria de fotos reais de projetos
* Botão flutuante de conversão direta para WhatsApp
* SEO otimizado para buscas locais no RJ
